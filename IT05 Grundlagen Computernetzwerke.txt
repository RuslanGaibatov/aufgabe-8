IT05 Grundlagen Computernetzwerke
🧠 1. Was ist ein Computernetzwerk?

Ein Computernetzwerk ist ein Zusammenschluss von Geräten (Computer, Smartphones, Server), die miteinander Daten austauschen.

👉 Ziele:

Daten teilen (z. B. Dateien)
Kommunikation (E-Mail, Chat)
Ressourcen nutzen (Drucker, Internet)
🌍 2. Netzwerkarten
LAN (Local Area Network) → kleines Netz (z. B. Schule, Zuhause)
WAN (Wide Area Network) → großes Netz (Internet)
WLAN → drahtloses LAN
🔌 3. Netzwerkgeräte
Router → verbindet Netzwerke (z. B. Internet)
Switch → verbindet Geräte im LAN
Access Point → WLAN-Zugang
Server → stellt Dienste bereit
Client → nutzt Dienste
📡 4. IP-Adresse

Jedes Gerät hat eine eindeutige Adresse.

Beispiel:

IPv4: 192.168.0.1

👉 Vergleich: wie eine Hausadresse

🌐 5. DNS (Domain Name System)

Übersetzt:

google.com → 142.250.xxx.xxx

👉 Menschen merken sich Namen, Computer brauchen Zahlen.

📦 6. OSI-Modell (sehr wichtig!)

7 Schichten:

Bitübertragung (Physical)
Sicherung (Data Link)
Vermittlung (Network)
Transport
Sitzung
Darstellung
Anwendung

👉 Merksatz:
"Alle Deutschen Studenten Trinken Verschiedene Sorten Bier"

🚚 7. TCP vs. UDP
TCP
sicher
langsam
überprüft Daten
UDP
schnell
unsicher
keine Kontrolle
🔐 8. Protokolle

Regeln für Kommunikation:

HTTP / HTTPS → Webseiten
FTP → Dateien
SMTP → E-Mails senden
DNS → Namen auflösen
📶 9. Datenübertragung
kabelgebunden (LAN)
drahtlos (WLAN, Bluetooth)
🔑 10. Sicherheit
Firewall → schützt Netzwerk
Verschlüsselung (HTTPS)
Passwortschutz
